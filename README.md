# Machine-Learning-Artificial-Intelligence-for-Data-Science
Repositório dedicado às atividades práticas da disciplina *Machine Learning & Artificial Intelligence for Data Science*. Este espaço reúne exercícios aplicados envolvendo análise exploratória de dados, modelagem preditiva e avaliação de modelos, com foco em problemas reais como séries temporais, correlação entre variáveis e seleção de features.

Ao longo do repositório, são exploradas técnicas de regressão, validação de modelos, engenharia de atributos (feature engineering) e análise de desempenho utilizando métricas como MAE. As atividades são desenvolvidas com Python e bibliotecas amplamente utilizadas no mercado, como pandas, scikit-learn e statsmodels.

O objetivo deste repositório é consolidar o aprendizado prático em ciência de dados, evidenciando a capacidade de transformar dados em insights e construir modelos preditivos com aplicação em cenários reais de negócio.
